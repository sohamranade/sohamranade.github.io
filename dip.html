Project Overview

Developed during the COVID-19 pandemic, this project implemented a real-time face mask detection system using convolutional neural networks. The system was designed to help ensure safety compliance in public spaces by automatically detecting whether individuals are wearing face masks.

Problem Statement

During the COVID-19 pandemic, face mask compliance became crucial for public safety. Manual monitoring was inefficient and prone to human error. This project aimed to create an automated solution that could:

  • Detect faces in real-time video streams
  • Classify whether a face mask is being worn
  • Provide immediate feedback and alerts
  • Maintain privacy while ensuring safety

Technical Approach

The system utilized a two-stage approach combining traditional computer vision techniques with deep learning:

  • Face Detection: Used OpenCV's Haar cascades for initial face detection
  • Mask Classification: Implemented a custom CNN for mask/no-mask classification
  • Real-time Processing: Optimized for live video stream processing
  • Multi-scale Detection: Handled faces at various distances and angles
Key Innovation: Custom CNN Architecture

Designed a lightweight convolutional neural network specifically optimized for mask detection. The architecture balanced accuracy with computational efficiency, enabling real-time performance on standard hardware while maintaining high detection accuracy.

Data Collection and Preprocessing

Created a comprehensive dataset for training the model:

  • Collected images from various sources with proper labeling
  • Applied data augmentation techniques (rotation, scaling, lighting changes)
  • Balanced dataset between mask and no-mask classes
  • Preprocessed images for optimal CNN training

Model Architecture

The CNN architecture consisted of:

  • Input Layer: 224x224x3 RGB images
  • Convolutional Layers: Multiple conv layers with ReLU activation
  • Pooling Layers: Max pooling for feature reduction
  • Dropout: Regularization to prevent overfitting
  • Output Layer: Binary classification (mask/no-mask)

Implementation and Results

The system achieved impressive performance metrics:

  • Accuracy: 95.2% on test dataset
  • Real-time Performance: 30 FPS on standard hardware
  • False Positive Rate: Less than 3%
  • Multi-face Detection: Simultaneous detection of multiple faces

The system successfully demonstrated robust performance across various lighting conditions, face angles, and mask types, making it suitable for real-world deployment.

Applications and Impact

This project had significant practical applications during the COVID-19 pandemic:

  • Public Safety: Automated monitoring in public spaces
  • Workplace Compliance: Office and factory safety monitoring
  • Research Contribution: Open-source implementation for community use
  • Educational Value: Demonstrated practical deep learning applications
